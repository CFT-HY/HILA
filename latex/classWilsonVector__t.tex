\subsubsection*{template$<$int Nvectors, int N, typename T$>$\newline
class Wilson\+Vector\+\_\+t$<$ Nvectors, N, T $>$}
Wilson\+\_\+vector contains the data for a single pseudofermion with the Wilson action. For each gamma dimension, contains a \doxylink{classSU}{SU(\+N)} vector. Wilson fermions can be multiplied by the gamma matrices gamma0, gamma1 ... gamma5 (or up to gamma 2 for 2 and 3 dimensions).

A gamma matrix can also be projected to a half\+\_\+\+Wilson\+\_\+vector using the constructor half\+\_\+\+Wilson\+\_\+vector(wilson\+\_\+vector, dir, sign). This multiplies the vector with 0.\+5\texorpdfstring{$\ast$}{*}(1-\/sign\texorpdfstring{$\ast$}{*}gamma\+\_\+dir), leaving half the degrees of freedom. A full Wilson vector can be recovered using half\+\_\+\+Wilson\+\_\+vector\+::grow.
