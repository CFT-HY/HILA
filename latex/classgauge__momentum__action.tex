class gauge\+\_\+momentum\+\_\+action$<$ gauge\+\_\+field $>$}
The action of the canonical momentum of a gauge field. Momentum actions are a special case. It does not contain a \doxylink{classaction__base_a478cf7430fc29604379676676cb8f656}{force\+\_\+step()}-\/function. It contains a \doxylink{classgauge__momentum__action_ae227beacf9d7ec638fbbca2d2a86fa05}{step()}-\/function, which updates the momentum itself. It can be used as the lowest level of an integrator.
