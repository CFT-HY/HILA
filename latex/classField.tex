\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Field$<$ T $>$}
something 

The field class implements the standard methods for accessing fields Hilapp replaces the parity access patterns, \doxylink{classField}{Field}\mbox{[}par\mbox{]} with a loop over the appropriate sites.

The \doxylink{classField}{Field} class also contains member functions used by hilapp, as well as members that may be useful for application developers.

The \doxylink{classField}{Field} mainly implements the interface to the \doxylink{classField}{Field} and not the content.

The \doxylink{classField}{Field} contains a pointer to Field\+::field\+\_\+struct, which implements MPI communication of the \doxylink{classField}{Field} boundaries.

The Field\+::field\+\_\+struct points to a \doxylink{classfield__storage}{field\+\_\+storage}, which is defined by each backend. It implements storing and accessing the \doxylink{classField}{Field} data, including buffers for storing haloes returned from MPI communication.
