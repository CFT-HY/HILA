class Dirac\+\_\+\+Wilson\+\_\+evenodd$<$ matrix $>$}
An even-\/odd decomposed Wilson Dirac operator. Applies D\+\_\+\{even to odd\} D\+\_\+\{diag\}\texorpdfstring{$^\wedge$}{\string^}\{-\/1\} D\+\_\+\{odd to even\} on the even sites of the vector.

The fermion partition function is det(\+D) = det(\+D\+\_\+eveneodd) + det(D\+\_\+\{diag odd\}). \doxylink{classDirac__Wilson__evenodd}{Dirac\+\_\+\+Wilson\+\_\+evenodd} can be used to replace D\+\_\+\+Wilson in the HMC action, as long as the diagonal odd to odd part is accounted for.

This is useful for defining inverters as composite operators. For example the conjugate gradient inverter is CG$<$\+Dirac\+\_\+\+Wilson\+\_\+evenodd$>$.
