\doxysubsection{Detailed Description}
Helper class for loading the vectorized lattice. 

Some backends need specialized lattice data in loops. These are implemented here. Loops generated by Transformer can access this through lattice.\+backend\+\_\+lattice.

Lattice related data that needs to be communicated to kernels
