handle an array subscript expression. Operations depend on whether the array is defined loop extern or intern:

  • if intern, nothing special needs to be done
  • if extern, check index:
    • if site dependent, mark loop_info.has_site_dep_cond_or_index
    • if contains loop local var ref or is site dependent: whole array is input to loop: need to know array size. If size not knowable, flag error
    • If index is not loop local:
      • it is sufficient to read in this array element only, and remove var references to variables in index
