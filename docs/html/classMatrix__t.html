Detailed Description

template<const int n, const int m, typename T, typename Mtype>
class Matrix_t< n, m, T, Mtype >

The main nxm matrix type template Matrix_t This is a root type, and "useful" types are derived from this type

Uses curiously recurring template pattern (CRTP), where the last template parameter is the template itself Example: the matrix type below is defined as template <int n, int m, typename T> class Matrix : public Matrix_t<n, m, T, Matrix<n, m, T>> { .. }

Used because stupid c++ makes it complicated to write generic code, in this case derived functions to return derived type
