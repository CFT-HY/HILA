This hook is called when #include (or #import) is processed. It adds the file to includelocs, a list of candidates that may need to be modified and inserted into the file buffer This is triggered when a pragma directive is encountered. It checks for the "#pragma hilapp" and stores the location, file and the command If pragma is "skip" it marks the tranlation unit for skipping.

Note that pragmas where the code location is important are handled in TopLevelVisitor::has_pragma().
