Detailed Description

template<typename T>
class Reduction< T >

Special reduction class: declare a reduction variable which can be used in site loops Example: Reduction<Complex<double>> rv = 0; onsites(ALL) rv += fv[X];

... // possibly do something not involving rv

Reduction can be modified to on/off: allreduce(), nonblocking(), delayed() Example: rv.allreduce(false).delayed();

Delayed reduction can be used e.g. in expressions like

Reduction<double> rv = 0; rv.allreduce(false).delayed(); foralldir(d) { onsites(ALL) rv += a[X+d]*a[X]; } rv.reduce();

This does only one reduction operation, not for every onsites() -loop. Result is the same

Reduction variable can be used again
