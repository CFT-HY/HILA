13 * The following vector classes are defined here:
14 * Vec64c Vector of 64 8-bit signed integers
15 * Vec64uc Vector of 64 8-bit unsigned integers
16 * Vec64cb Vector of 64 booleans for use with Vec64c and Vec64uc
17 * Vec32s Vector of 32 16-bit signed integers
18 * Vec32us Vector of 32 16-bit unsigned integers
19 * Vec32sb Vector of 32 booleans for use with Vec32s and Vec32us
20 * Other 512-bit integer vectors are defined in Vectori512.h
21 *
22 * Each vector object is represented internally in the CPU as a 512-bit register.
