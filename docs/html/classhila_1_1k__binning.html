Detailed Description

class hila::k_binning is used to bin "k-space" fields (typically fourier transformed from real space). Vector k is normalized so that -pi < k_i <= pi, i.e. coordinate. Bin is determined by formula b = (int) ( pow(k/k_max)^p * n_bins )

hila::k_binning kb; kb.bins(80).k_max(M_PI); auto sd = kb.spectraldensity(f); // get the spectral density of field f

methods: (use as kb.xxx) hila::k_binning & bins(int n) set number of bins int bins() get number of bins hila::k_binning & k_max(double m) set max value of k in binning double k_max() get max value of k hila::k_binning & power(double p) set the "power" in binning double power() get power

std::vector<T> bin_k_field(const Field<T> &f) bin the k-space field f std::vector<double> bin_k_field_squarenorm(const Field<T & f) bin the square norm of k-space field f std::vector<double> spectraldensity(const Field<T> &f) FFT real-space field f and bin the result in squarenorm
