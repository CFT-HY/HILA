Detailed Description

template<typename T>
class SU2< T >

This implementation represents the matrices as $U = d + a i\sigma_1 + b i\sigma_2 + c i\sigma_3$ This is in SU(2) if $a^2 + b^2 + c^2 + d^2 = 1$

SU2 matrices are equivalent to SU<2, radix>, but are implemented more efficiently here. This implementation represents the matrices as U = d + a * sigma_1 + b * sigma_2 + c * sigma_3. This is in SU(2) if a^2 + b^2 + c^2 + d^2 = 1.
